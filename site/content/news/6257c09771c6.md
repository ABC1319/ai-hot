腾讯云的企业级智能体平台，正式出海了。

7月18日，在2026世界人工智能大会上，腾讯云正式发布了智能体开发平台 ADP 4.0海外版，同步升级智能工作台、Claw 模式、Skill 广场三大核心模块，围绕触达、交互、生态、连接四大能力做了全面国际化适配。

ADP 的全称是 Agent Development Platform，定位为企业级 AgentOps 平台，覆盖智能体的构建、分发和治理全生命周期。

简单来说，它解决的是企业智能体从建立、运行到管理的问题。

智能体开发平台 ADP 4.0海外版
这次企业级智能体海外版的核心升级集中在四个方面：一是渠道，Claw 应用支持 LINE、Telegram 等主流海外社交渠道，一站发布即可跨区域触达全球用户；
而在交互体验上，智能工作台支持自定义时区与多区域调度，模型输出可跟随用户输入语言自动响应；
在开放生态上，模型广场已经接入多家国际主流大模型，平台 Skill 完成国际化适配；并且，平台深度接入 Google Workspace、Confluence、Jira 等海外主流 SaaS 应用。

ADP 4.0的一个关键能力升级是 Agent 与 Workflow 双向互调。

吴运声在采访中解释了这个设计的逻辑：企业在实际使用中，有些事情是模糊的、开放的，适合用 Agent 方式处理，这会消耗较多 Token；但一些确定性的流程，可以用 Workflow 来做，不需要反复消耗 Token。

两者结合，既满足业务需求，又降低成本。

这套逻辑在国内也已经有了标杆案例。

比如，找钢网基于 ADP 4.0 Claw 模式搭建了企业中台数字员工，接入企业微信，覆盖订单、物流、财务、仓库、发票等多类场景。

Agent 负责理解开放问题，Workflow 负责执行严谨业务流程，两者在同一个应用中协同工作。
