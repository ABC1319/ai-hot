GMI Cloud 携AI Cloud、MaaS、Agentbox等全栈智算解决方案亮相 WAIC 2026
量子位的朋友们
2026-07-20
12:34:07
来源：
量子位
作为全球七大 Reference Platform NVIDIA Cloud Partner 之一，GMI Cloud 携全栈产品矩阵、前沿技术成果及丰富联合生态，在 2026 年世界人工智能大会（WAIC）上通过产品展示、专家分享、跨界联名及直播互动等多元形式，与全球开发者、AI 企业及生态伙伴共探 AI 算力新未来，成为展会焦点之一。

在本次大会上，GMI Cloud 集中展示了四大核心产品，为观众打造了沉浸式的 AI 基础设施交互体验。

GMI Cloud 作为领先的“AI Native Cloud”，凭借全球布局的自建数据中心，提供基于 GB300、GB200、B200 及 H200 等全阵容高性能芯片的 GPU 云服务，为全球企业量身打造稳定安全、高效经济的 AI 基础设施。

目前，GMI Cloud 已在中国台北构建基于 GB300 的万卡集群“AI Factory”。

此外，GMI Cloud Cluster Engine （Compute Engine） 作为 GMI Cloud 面向 AI 应用自主研发的云原生算力管理调度平台，包含了裸金属与容器算力服务，让算力资源调配更加敏捷高效。

Inference Engine 则是全球 AI 模型统一接入与在线使用的“高性能推理引擎平台”，底层搭载顶配芯片并集成全球前沿大模型；而 Agentbox 平台则打通了 AI Agent 全链路部署与商业化落地的最后一公里。

在本届 WAIC 上，GMI Cloud 展位特别设置的产品体验区，其自研工具直击 AI 应用构建与调用的痛点。

参会者无需复杂流程，现场就能借助 GMI Cloud 自研的 Inference Engine，在平台上极速调用全球超 200 种主流大模型 API，沉浸式感受在海外快速落地 AI 想法的高效体验，助力开发者加速创新进程。
