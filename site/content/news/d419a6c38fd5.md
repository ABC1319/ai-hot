最新发布的
Android Studio Quail 2
现已进入稳定版。

该版本扩展了 IDE 内 Gemini/AI Agent Mode 的能力，支持多个 AI 对话并行运行，并进一步推进 Google 将 AI 驱动工作流直接融入 Android Studio 的战略。

同时，该版本还增强了调试和性能分析能力，并让开发者更容易探索实验性功能。

Google 表示，Quail 2 中的 Agent Mode 已经过全面重新设计，带来了更好的性能、更有效的任务拆解能力，以及更完善的工具支持。

其中最值得关注的是，新设计允许开发者同时处理多个 Agent 对话：
过去，开发者必须等待 Android Studio Agent 完成任务后，才能在 Agent Mode 中提出另一个问题或启动独立任务，这种瓶颈已经成为过去。

现在，你可以无缝进行多任务处理：在一个标签页中启动 UI 重构，在第二个标签页中修复 ProGuard 规则，在第三个标签页中生成文档。

Android Studio 的 Agent Mode 除了支持 Gemini 之外，还支持多个大型语言模型。

为了帮助开发者了解哪些模型在特定任务中表现更好，Google 创建了专门针对 Android 开发的基准测试
Android Bench
。

该基准测试会使用一组常见开发任务对模型进行评估，同时将权限、导航、连接等领域的 Android 最佳实践纳入考量。

Android Studio Quail 2 还通过将广受欢迎的开源工具 LeakCanary 集成到 IDE 中，提升了对
内存泄漏检测
