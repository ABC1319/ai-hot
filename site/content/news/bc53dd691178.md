7 月 6 日，腾讯正式发布混元 Hy3。

该模型是 Hy3 preview 的正式版本，重点提升了智能体（Agent）、代码生成、办公任务和复杂推理等能力，并进一步下调调用价格。

Hy3 的 API 定价为：1 元/百万 tokens，输出 4 元/百万 tokens，输入命中缓存价格 0.25 元/百万 tokens。

。

腾讯同时宣布，Hy3 将采用 Apache 2.0 协议开源。

开发者可以下载模型并用于商业场景。

目前，Hy3 已接入 WorkBuddy、CodeBuddy、腾讯元宝、Marvis、ima 等腾讯内部产品，API 已在腾讯云 TokenHub 上线，后续还将接入多个海外 API 平台，覆盖 OpenRouter、Hermes、Kilo、Cline、OpenClaw、OpenCode、CherryStudio 等，且同时“day 0”接入开源模型社区 Huggingface、Modelscope 魔搭平台。
