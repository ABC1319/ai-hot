HeroUI
（前身为 NextUI）是一款 React 组件库，现在
已发布 HeroUI v3 版本
。

该版本经过了彻底的重写，开发团队将其定位为适用于 Web 和原生应用的统一设计系统。

这个于 2026 年 3 月发布的版本包含超过 75 个 Web 组件（其中 21 个为新增组件），同时还提供了一个从零开始构建的全新的 React Native 库，内含 37 个组件。

HeroUI v3 基于
React Aria Components
实现了无障碍功能，并采用
Tailwind CSS v4
进行了样式设计，同时为方便自定义，提供了 CSS 变量、OKLCH 颜色和 BEM 修饰符。

所有动画均已迁移至 CSS，不需要 JavaScript 运行时支持。

该库采用了复合组件 API，摒弃了旧版的 provider 包装器，并且可完美兼容 React 19 和 Next.js 。
